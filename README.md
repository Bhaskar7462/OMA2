# OMA2

OMAA (Ollama Multi Agent Assistant) is a local AI assistant built using Python, Ollama, and a multi-agent architecture.

## Features

- Local LLM using Ollama
- Multi-agent architecture
- Weather agent
- Search agent
- System agent
- Application launcher
- Modular tool system

## Tech Stack

- Python
- Ollama
- AutoGen (future integration)
- OpenWeather API
- Linux

## Project Structure

agents/
config/
tools/
main.py
